# 🎮 Agent IA - Puissance 4 en Python

Projet Python implémentant le jeu **Puissance 4** avec une architecture orientée objet.
Le projet est conçu pour évoluer vers l’ajout d’un **agent intelligent (IA)**.

---

## 📌 Objectifs du projet

- Implémenter le jeu Puissance 4 en Python
- Utiliser la programmation orientée objet
- Structurer proprement un projet Python
- Ajouter progressivement une IA capable de jouer contre un humain

---

---

## 🧠 Fonctionnalités actuelles

✅ Plateau 6x7  
✅ Gestion des tours joueurs  
✅ Vérification des coups valides  
✅ Détection de victoire (horizontale, verticale, diagonale)  
✅ Gestion des erreurs utilisateur
